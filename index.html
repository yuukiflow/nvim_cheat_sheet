Neovim Keymap Cheat Sheet

Normal Mode

  • Space pv : Open file explorer
  • h : Move to the left window
  • j : Move to the bottom window
  • k : Move to the top window
  • l : Move to the right window
  • b : Move to the beginning of the line
  • e : Move to the end of the line
  • J : Join lines and keep cursor in place
  • d : Scroll down and center the screen
  • u : Scroll up and center the screen
  • n : Go to the next search match and center the screen
  • N : Go to the previous search match and center the screen
  • Q : Disable Q in normal mode
  • f : Open a new tmux window in silent mode
  • Space f : Format using LSP
  • Space k : Go to the next location list item and center
  • Space j : Go to the previous location list item and center
  • Space s : Search and replace word under cursor globally
  • Space x : Make the current file executable
  • Space vpp : Open packer configuration file
  • Space mr : Run CellularAutomaton plugin
  • ; : Replace ; with : for commands in normal mode
  • Space Space : Source the current file

Insert Mode

  • h : Move the cursor left
  • j : Move the cursor down
  • k : Move the cursor up
  • l : Move the cursor right
  • s : Save the file
  • c : Escape insert mode
  • F1 : Accept codeium suggestion

Visual Mode

  • J : Move selected text down
  • K : Move selected text up
  • Space y : Yank to system clipboard
  • Space d : Delete without copying to register
  • Space p : Paste over and delete the selected text

Visual + Normal Mode

  • Space y : Yank to system clipboard
  • Space d : Delete without copying to register
